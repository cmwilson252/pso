How the Table Works

The lists contain purely quests that people TA. If there are any missing let me know. The values for each warp are in the order of the quest progresses. So for instance MA1C goes from Cave 2 -> Mines 2 -> Ruins 3 -> Dark Falz which has warp values of 4, 7, 10, 14.
