PSV: TA Teamz Player Cards

Vanilla Commands

  • /tech
    Maximum level Techniques are learned.

  • /reset
    Character level set to lv.1, and all materials except HP/TP are reset (can't be used in lobby or P2).

  • /matcount
    Displays used Pow/Mind/DFP/EVP/LCK materials.

  • /$
    Gives maximum meseta (cannot use mid quest).

  • /scape
    Adds one Scape Doll to inventory (cannot use mid quest).

  • /moon
    Restores Moon atomizers (cannot use mid quest).

  • /monomate
    Restores Monomates (cannot use mid quest).

  • /pipes
    Restores telepipes (cannot use mid quest).

  • /mates
    Restores Dimate, Trimate, Moon Atomizer, and Sol Atomizer (cannot use mid quest).

  • /fluids
    Restores Difluid, and Trifluid (cannot use mid quest).

  • /drop
    Re-enables drops. Use the command again if you want to deactivate drops.

  • /rq
    Leave quest at anytime.

  • /name " "
    Changes character name. Need to change blocks to see changes.

  • /heal
    Heals HP by 2,400 (cannot use mid quest).

  • /lower
    Lowers HP value by 1024 (cannot use mid quest).

  • /id " "
    "viridia"
    "greennill"
    "skyly"
    "bluefull"
    "purplenum"
    "pinkal"
    "redria"
    "oran"
    "yellowboze"
    "whitill"
    After selecting a Section ID you must change blocks for changes to be made.

  • /matplan
    Automatically applies the Min-Max material plans.

  • /gear
    Spawns TA ready gear based on your class. It is sent to your bank.
